## Getting Started

To get a local copy up and running follow these simple example steps:

### Pre-requisites:

* npm

```sh
     npm install npm@latest -g
```

### Installation

* Clone the current repository

* Install NPM packages

```sh
     npm install
```

* Run command inside root folder

```sh
     npm run dev
```
